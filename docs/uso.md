# Uso Básico

Aprende a realizar las operaciones fundamentales en tu flujo de trabajo diario.

## Ciclo de Vida

El ciclo de vida en este framework se basa en iteraciones cortas.

1.  **Planificación**: Definir los objetivos del ciclo.
2.  **Desarrollo**: Ejecutar las tareas.
3.  **Revisión**: Demostrar el progreso.

!!! warning "Atención"
    No saltes la fase de planificación, es crucial para el éxito del equipo.
