# Instalación y Configuración

Para comenzar a trabajar con nuestro framework ágil, sigue estos pasos de instalación.

## Requisitos Previos

Asegúrate de tener instalado Python y un editor de código.

=== "Windows"

    ```powershell
    pip install agile-toolset
    agile-toolset --init
    ```

=== "macOS / Linux"

    ```bash
    pip3 install agile-toolset
    agile-toolset --init
    ```

!!! tip "Consejo"
    Recomendamos usar un entorno virtual para evitar conflictos de dependencias.
